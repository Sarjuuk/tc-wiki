> The data in this table is maintained by the core.
{.is-info}

| Field | Type | Attributes | Key | Null | Default | Extra | Comment |

| [numchars](#numchars) | tinyint(3) | unsigned |  | NO | 0 |  |  |

## Description of fields

### realmid
The ID of the realm.
&nbsp;

### acctid
The account ID.
&nbsp;

### numchars
