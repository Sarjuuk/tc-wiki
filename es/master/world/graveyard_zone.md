## Description of fields

### ID
WorldSafeLocs ID, see [world_safe_locs](https://dev.trinitycore.info/en/database/master/world/world_safe_locs) table
&nbsp;

### GhostZone
ZoneId before teleportation to graveyard (before pressing release button), see AreaTable.db2 or use .gps command ingame while within the zone.
&nbsp;

### Faction
Graveyard's team

0 - Any team accepted
469 - Alliance team only
67 - Horde team only
